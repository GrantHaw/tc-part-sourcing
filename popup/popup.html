    How to use

    1. Open an eBay or Amazon listing for a candidate part.

    2. Click the extension icon. Fields pre-fill from the page.

    3. Verify every field — parsers miss things.

    4. Hit Add to list. Repeat for each candidate.

    5. Reorder with arrows so Option 1 is your best pick.

    6. Click Copy all and paste into the ticket.

    7. Click Clear list before starting the next ticket.

    Installing (unpacked)

    1. Unzip the extension folder somewhere permanent.

    2. Go to chrome://extensions.

    3. Toggle Developer mode on (top right).

    4. Click Load unpacked and select the partsourcer folder.

    5. Pin the icon to your toolbar so it's one click away.

    Watch out for

    • eBay and Amazon change their layouts regularly — if fields come back blank, the parser is stale. Flag it so selectors can be updated.

    • If scraping does nothing on a tab that was open before you installed, reload the page first.

    • Rating format differs: eBay shows 99.2% positive, Amazon shows 4.5/5 stars. Both are fine.

    • Options are stored locally only — switching machines mid-ticket loses the list.

    • On any non-eBay/Amazon page the form opens blank except for the URL. Fill it in by hand.

    • If the detached window is open and you also click the toolbar icon, you'll get both. Close whichever you don't want.
